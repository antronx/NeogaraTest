Проект написан на Java с использованием Selenium Chrome Driver, JUnit 5, Allure. API тесты писались с использованием библиотеки Rest Assured. 

*****************************************
**************UI тесты*******************
*****************************************

Структура проекта:
1. Класс com/test/app/TestForm.java - содержит элементы тестовой страницы, а также действия, которые можно выполнять с элементами страницы. 
В том числе считывание текстов и взаимодействие с алертами
2. Класс com/test/app/TestWatcher.java - класс делает скриншот при падении теста, и сохраняет его в директорию \screenshots.
	Проблема: Так как алерты не являются частью страницы, они не попадают на скриншот. toastr сообщения - попадают
3. Класс com/test/app/UITests.java - В этом классе хранятся все написанные тесты.

Чтобы запустить тесты в косоли IDE нужно выполнить следующие команды:
1. mvn clean test -Dtest=UITests.java
2. После того как тесты прошли можно посмотреть результаты при помощи Allure, для этого нужно выполнить команду:
mvn allure:serve
В браузере откроются результаты тестов. Там же можно посмотреть скриншот в на момент падения теста.
